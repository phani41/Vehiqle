## 📋 About The Project

Vehiqle (also known as TruckVerse) aims to solve the complexities of logistics and vehicle fleet management. This platform provides a user-friendly interface for tracking, managing, and analyzing fleet data, helping businesses optimize their operations and reduce costs.

Whether you're an independent owner-operator or managing a large fleet, Vehiqle is designed to be your digital co-pilot.

## 🏁 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm (or yarn/pnpm) installed on your machine.
*   **npm**
    ```sh
    npm install npm@latest -g
    ```

### Installation

1.  **Clone the repository**
    ```sh
    git clone https://github.com/Saiphanikum/Vehiqle.git
    ```
2.  **Navigate to the project directory**
    ```sh
    cd Vehiqle
    ```
3.  **Install NPM packages**
    ```sh
    npm install
    ```
4.  **Run the development server**
    ```sh
    npm run dev
    ```
5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
